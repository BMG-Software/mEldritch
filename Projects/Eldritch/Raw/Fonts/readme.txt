This font and description file was used to generate the replacement font for
the level editor. Call:

  mFontGenerator freesans10.font freesans10.fnp freesans10.tga Textures/Fonts

to generate the files in the current directory, and have them prepared for
the textures to go to the Textures/Fonts directory. The fnp file should be
placed in the Fonts directory.
